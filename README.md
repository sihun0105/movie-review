## Description

Nest를 사용한 영화평론 프로젝트입니다.
- 주간 탑10개의 영화에 대하여 사용자들의 자유로운 평론이 가능합니다.
- 영화정보는 [한국영화진흥원](https://www.kobis.or.kr/kobisopenapi/homepg/main/main.do)의 open APi 를 활용하여 "하루 1회" 데이터를 업데이트 합니다.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
